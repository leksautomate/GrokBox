# GrokBox — Chrome Web Store Publishing Materials

## Required Assets Checklist
- [x] Extension icon (128x128 PNG) — `icons/icon128.png`
- [x] Store listing description — `store/description.txt`
- [x] Privacy policy — `store/privacy-policy.md`
- [x] Screenshot generator — `store/screenshots.html`
- [ ] Small promo tile 440x280 — capture from `store/promo.html`
- [ ] Screenshots 1280x800 — capture from `store/screenshots.html`

## How to Capture Screenshots
1. Open `store/screenshots.html` in Chrome
2. Press F12 → Device toolbar → set to 1280x800
3. Take screenshots of each slide
4. Save as PNG files

## How to Capture Promo Tile
1. Open `store/promo.html` in Chrome
2. Screenshot at 440x280
